## 安装

使用 `pnpm`

```bash
pnpm add @tomjs/electron-devtools-installer
```

使用 `yarn`

```bash
yarn add @tomjs/electron-devtools-installer
```

使用 `npm`

```bash
npm i @tomjs/electron-devtools-installer
```
